# CPI India Forecasting with SARIMA

## 📊 Overview

This project forecasts India's **CPI Combined Index (All India)** using SARIMA (Seasonal ARIMA) time series models. The analysis spans from January 2013 to November 2025, with a 12-month out-of-sample forecast.

**Data Source:** Reserve Bank of India - Database on Indian Economy (RBI DBIE)
**Base Year:** 2012 = 100

## 🎯 Key Results

| Metric | Value |
|--------|-------|
| **MAPE** | 2.60% |
| **MAE** | 5.07 |
| **RMSE** | 5.49 |
| **Training Period** | Jan 2013 - Nov 2024 (143 months) |
| **Testing Period** | Dec 2024 - Nov 2025 (12 months) |
